# BIG O 
## Linear Search BIG O
- Best O(1)
- Average O(n)
- Worst O(n)

## Binary search BIG O?
- Best Case O(1) - could be first guess, lucky middle shot.
- Worst and Average Case O(log n) 

Naive String Search - O(nm)
KMP - O(n + m) time, O(m) space

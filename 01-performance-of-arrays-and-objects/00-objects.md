# Objects
Unordered, key value pairs !
```cs
let instructor = {
    firstName = "Kelly",
    isInstructor = true,
    favNums: [1,2,3,4]
}

```

## When to use Objects ?
- when you don't need order
- when you need fast access/insertion and removal

## Big O of Objects
- Insertion - O(1)
- Removal - O(1)
- Seaching - O(n)
- Access - O(1)

## Big O of Object Methods
- Object.keys - O(n)
- Object.values - O(n)
- Object.entries - O(n)
- Object.hasOwnProperty - O(1)
